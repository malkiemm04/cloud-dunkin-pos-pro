# Cloud Dunkin' POS Pro ☁️

A cloud-native Point of Sale system for Dunkin' Donuts built with AWS serverless technologies.

## Features
- ✅ Cloud-hosted POS system with HTTPS
- ✅ Serverless backend (AWS Lambda)
- ✅ Managed database (AWS DynamoDB)
- ✅ CDN caching via CloudFront
- ✅ User authentication with Cognito
- ✅ Real-time monitoring with CloudWatch
- ✅ Cost control with AWS Budgets
